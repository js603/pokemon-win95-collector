# Pokemon Win95 Collector (Retro Emulator)

Windows 95 테마와 공식 PokéAPI를 탑재한 1세대 포켓몬 퀴즈 수집 게임입니다.

## 로컬 실행 방법 (How to Run Local)

1. 이 압축 폴더(`pokemon-win95-collector.zip`)를 원하는 위치에 압축 해제합니다.
2. 해당 디렉토리로 터미널(Terminal)을 이동합니다.
3. 패키지 의존성 모듈을 설치합니다:
   ```bash
   npm install
   ```
4. 로컬 개발 테스트 서버를 구동합니다:
   ```bash
   npm run dev
   ```
5. 브라우저에서 화면에 표시된 로컬 URL (`http://localhost:5173`)로 접속하여 레트로 포켓몬 게임을 마음껏 플레이하세요!
